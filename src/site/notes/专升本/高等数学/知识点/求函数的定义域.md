## 常考函数
1. $y = \frac{1}{2} : x \ne 0$ ;
2. $y = \sqrt{x} : x \ge 0$ ;
3. $y = log_ax (lnx, lgx):x > 0$ ;
4. $y = arcsinx$ 或 $arccosx : -1 \le x \le 1$ ;
5. $y=tanx:x\ne\frac{π}{x}+kπ$ ;  $y=cotx:x\ne kπ$ .
## 解题方法
解不等式（组）
