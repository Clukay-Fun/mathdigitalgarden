> [!important] #重要 
>$y = \frac{1}{2} : x \ne 0;$ 
>
>$y = \sqrt{x} : x \ge 0;$ 
>
>$y = log_ax (lnx, lgx):x > 0 ;$
>
>$y = arcsinx$ 或 $arccosx : -1 \le x \le 1 ;$
>
>$y=tanx:x\ne\frac{π}{x}+kπ$ ;  $y=cotx:x\ne kπ .$

> [!warning] 注意 #常见错误 
> 定义域是集合的形式，答案要写成==集合或区间的形式==
